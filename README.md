# Bike Sharing Systems - Demand Analysis

An end-to-end data analysis project exploring global bike sharing systems
and demand forecasting for Seoul, using R.

## Pipeline
1. Data collection via web scraping and API calls
2. Data wrangling with Tidyverse and regular expressions
3. Exploratory data analysis with SQL and ggplot2
4. Demand forecasting with linear regression via Tidymodels
5. Interactive dashboard built with R Shiny

## Stack
R, Tidyverse, ggplot2, Tidymodels, R Shiny, rvest, httr
